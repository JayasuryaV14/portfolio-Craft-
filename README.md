🚀 Portfolio Craft

📌 Overview

Portfolio Craft is a personalized portfolio generation platform that helps users create professional portfolios based on their own data — **not AI-generated, but fully customized** according to user inputs.

Unlike generic portfolio builders, Portfolio Craft focuses on **manual personalization**, ensuring each portfolio reflects the user's individuality and style.

---

✨ Key Features

🎨 Template Selection

  * Users can choose from:

    * Normal Templates
    * Premium Templates

📝 Personalized Portfolio Creation

  * Users provide their details through a form
  * Templates are customized manually based on user input

📄 Resume Upload

  * Users can upload their resume (via form submission)

📦 Portfolio Delivery

  * Users receive:

    * 🌐 Hosted portfolio link
    * 📁 Complete project files (ZIP format)

 🤖 AI Chatbot Support

  * Built-in chatbot to clarify user doubts and provide assistance

---

 🖥️ Project Structure

The platform consists of the following main pages:

* 🏠 Home Page
* 🔐 Login Page
* 📝 Sign Up Page
* 🎨 Template Selection Page

  * Normal Templates
  * Premium Templates
* 👀 Template Preview Option
* 📋 User Input Form (Google Form Integration)
* 🤖 Chatbot Page

---

⚙️ How It Works

1. User signs up or logs in
2. User explores available templates
3. User selects a template (Normal / Premium)
4. User fills in personal details via form
5. User uploads resume
6. Portfolio is customized based on the provided data
7. Final output is delivered:

   * Live hosted portfolio link
   * Downloadable ZIP file
